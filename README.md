# FH6Auto-RU

**Инструмент на Python для визульной автоматизации Forza Horizon 6.**  
**Распознаёт экран, имитирует действия. Поддерживает:** фарм гонок, автопокупку машин, прокачку навыков, удаление авто из гаража, объединение в сценарии и бесконечный цикл.

> **Важно:** Инструмент создан для изучения Python-автоматизации и технического экспериментирования. Не используйте его для получения коммерческой выгоды или нечестного преимущества.

## 📄 О проекте

**FH6Auto-RU** — это десктоп-утилита, которая управляется через анализ изображения на экране.

**В отличие от простых макросов, которые слепо жмут по координатам, FH6Auto-RU:**

- **Делает скриншоты** и анализирует, что сейчас происходит в игре
- **Динамически принимает решения**: зашло ли меню, найдена ли нужная машина, не зависла ли загрузка
- **Управляет мышью и клавиатурой**, имитируя действия живого игрока
- **Пытается восстановиться** при ошибках (закрыть аварийное окно, вернуться в меню)

Это делает скрипт гораздо более **стабильным, гибким и надёжным**, чем тупой кликер по координатам.

## ⚙️ Возможности (модули)

### 1️⃣ Цикл гонок (фарм очков/денег)
- Сам открывает нужные меню
- Переходит в раздел **«Творческий центр» / EventLab**
- Вводит **код пользовательской трассы**
- Выбирает нужную машину
- Запускает гонку и повторяет её нужное количество раз

### 2️⃣ Пакетная покупка машин (для прокачки)
- Заходит в **«Мои машины» → «Автосалон»**
- Находит машину по бренду и модели
- Покупает её указанное количество раз

### 3️⃣ Автопрокачка (Super Wheelspin фарм)
- Покупает/выбирает машину-донора
- Открывает её дерево **«Мастерство автомобиля»**
- Тратит **Очки навыков (Skill Points)** по заданному тобой маршруту, чтобы добраться до **Super Wheelspin'а** и забрать его
- Автоматически останавливается, когда очки навыков заканчиваются

### 4️⃣ Удаление машин
- Автоматически удаляет машины-доноры из гаража
- Освобождает место для новых покупок

### 5️⃣ Большой цикл (бесконечный конвейер)
Выстраивает **полный цикл фарма** из трёх предыдущих модулей:

**🏁 Фарм гонок (набор SP) → 🛒 Покупка машин → 🎁 Прокачка и открытие Super Wheelspin'ов → 🗑️ Удаление доноров → Сброс счётчика**

**Гибкие настройки:**
- Переходить или нет к следующему модулю
- Повторять ли большой цикл после завершения
- Количество повторений цикла (можно выставить очень много — получится «бесконечный» фарм)

## 🛠 Как пользоваться

### 1️⃣ Подготовка перед запуском

#### 🚗 Машина для фарма гонок (основная)
Вам понадобится **Subaru Impreza 22B-STi Version**. Сделайте с ней следующее:
- Поставьте тюнинг до индекса **S2 900**
- Добавьте автомобиль в **«Избранное»**
- На ней **не должно быть нанесений (ливреи/покраски)**

#### ✅ Проверьте эти условия
- **Отключите** любые фильтры изображения, HDR и прочие эффекты, влияющие на цветопередачу
- Игра **запущена** и активна
- **Язык игры: Русский**
- Раскладка клавиатуры в Windows: **ENG**
- В настройках сложности игры: **Авторулевое**, **Автокоробка**, сложность можно любую (например, **«Новичок»**)
- **Не переключайтесь** во время работы скрипта на другие окна, чтобы не сбивать распознавание

> **⚠️ ВАЖНО:** Скрипт распознаёт кнопки по картинкам-шаблонам.  
> Для работы с **русской версией игры** вам нужно заменить шаблоны в папке `images` (подробнее в разделе «Замена шаблонов» ниже).

### 2️⃣ Настройка параметров
В главном окне программы задайте:
- Количество заездов, покупок, прокачек
- **Код трассы** из EventLab
- Количество больших циклов
- Опции: «продолжить дальше», «зациклить», включить авто-перезапуск игры (тест)

### 3️⃣ Настройка пути прокачки (Дерево навыков)
В разделе **«Супер-розыгрыш / Прокачка»**:
- С помощью **кнопок со стрелками** выкладывается путь (сетка 4×4), который должен пройти скрипт в дереве мастерства
- **Синие клетки** — это шаги скрипта
- **«Очистить матрицу»** — сбросить путь

### 4️⃣ Запуск отдельного модуля
Можно запускать любую вкладку («Цикл гонок», «Покупка», «Прокачка», «Удаление») независимо.

### 5️⃣ Запуск цепочки (конвейера)
Если поставить галочки **«Продолжить»** между модулями, они будут запускаться друг за другом автоматически.

### 6️⃣ Остановка скрипта
- Нажать кнопку **«Стоп»** в интерфейсе
- Или нажать клавишу **F8** на клавиатуре

Скрипт попытается аккуратно всё завершить, отпустить зажатые кнопки и вернуть управление.

## 🖼️ Замена шаблонов изображений (ВАЖНО!)

Скрипт распознаёт кнопки по картинкам-шаблонам. По умолчанию в папке `images` лежат шаблоны **для китайской версии игры**.

**Для работы с РУССКОЙ версией игры вы ДОЛЖНЫ заменить их на скриншоты кнопок из вашей игры!**

### Основные шаблоны:

| Имя файла | Что должно быть на скриншоте |
|-----------|------------------------------|
| `CCbrand.png` | Логотип или кнопка для входа в раздел покупки машин |
| `consumablecar.png` | Иконка/область конкретной машины-донора (Subaru) в меню покупки |
| `newCC.png` | Маркер «новая машина» (обычно жёлтая звёздочка или значок) |
| `skillcar.png` | Иконка вашей **рабочей Subaru** в гараже (с отметкой «Избранное») |

### Как сделать правильные шаблоны:

1. **Запустите игру на русском языке**
2. Сделайте скриншот **только нужной кнопки/иконки** (не всей игры)
3. Обрежьте изображение так, чтобы на нём был только нужный элемент
4. Сохраните с **точным именем** (например, `CCbrand.png`) в папку `images` (заменив китайские шаблоны)
5. Убедитесь, что размер шаблона не слишком большой — скрипту будет проще искать

### Как это работает:
1. Скрипт сначала ищет шаблоны в папке `images` рядом с собой
2. Если их там нет — использует встроенные (китайские)
3. Если вы заменили шаблоны — скрипт будет искать русские кнопки

**Это позволяет адаптировать скрипт под любое разрешение, язык или даже визуальный стиль игры.**

## ⚡ Защита от ошибок и восстановление

Скрипт постоянно проверяет, всё ли идёт по плану. Если происходит ошибка:
- Не может найти кнопку
- Интерфейс завис
- Игра вылетела

**FH6Auto-RU попытается:**
1. Проверить, запущен ли процесс игры
2. Вернуть фокус на окно игры
3. Нажать кнопки «Назад», «В меню», чтобы выйти в безопасное место
4. Попробовать выполнить шаг снова
5. **(Тестовая фича)** Если игра закрылась — запустить её заново по указанной команде и продолжить работу

**Также при принудительной остановке (F8) скрипт гарантированно «отпустит» все зажатые клавиши** (W, Shift, Enter и т.д.), чтобы вы не остались с «залипшим» управлением.

## 💡 Ключевые особенности (почему это круто)

- **Умная автоматизация, а не тупой кликер**  
  Использует `opencv` для распознавания образов в реальном времени

- **Не «слепой» метод**  
  Перед каждым действием проверяет, то ли окно активно, и та ли кнопка сейчас на экране

- **Стабильность**  
  Не ломается, если игра чуть подвисла или загрузка идёт на секунду дольше

- **Гибкая настройка**  
  Поддерживает замену шаблонов, настройку пути прокачки, разные сценарии запуска

- **Безопасная остановка**  
  Всегда отпускает все клавиши, не оставляя виртуальный «залипший» газ или руль

- **Бесконечный конвейер**  
  Собирает из простых модулей сложный цикл фарма, который может работать часами

## ⚙️ Технологии

Проект построен на современном стеке Python:

| Компонент | Назначение |
|-----------|------------|
| `customtkinter` | Удобный и красивый интерфейс |
| `opencv-python (cv2)`, `numpy` | Распознавание образов, сравнение изображений |
| `pyautogui`, `pydirectinput` | Управление экраном, скриншоты, эмуляция мыши и клавиатуры |
| `threading` | Фоновая работа (чтобы интерфейс не зависал) |
| `PIL` | Работа с картинками |
| `win32gui / ctypes` | Системные вызовы Windows (фокус окна, снятие клавиш) |
